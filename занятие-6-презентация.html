ПОДХОД 1 Kimi Slides
Пишешь тему → получаешь готовый .pptx за 60 секунд
Плюсы
• 37 шаблонов, русский язык
• Бесплатно, без водяных знаков
• .pptx — текст и элементы редактируемые
• Outline перед генерацией
Как использовать
1. Открыть kimi.ai → Slides
2. Ввести тему на русском
3. Выбрать шаблон
4. Скачать .pptx
Также: Qwen — рендерит слайды как картинки 2048×2048 (красиво, но не редактируемо)

ПОДХОД 2 NotebookLM — Slide Deck
Загрузили документы → получили презентацию на основе ваших данных
Суперсила
• Точность: контент из ваших документов
• Нет галлюцинаций — только ваши данные
• Настройки: Detailed / Presenter
• Per-slide revision через промпт
Как использовать
1. notebooklm.google.com
2. Загрузить PDF / Google Doc / сайт
3. Studio → Slide Deck
4. Настроить стиль через промпт
5. Экспорт в PDF или .pptx
Вы уже работали с NotebookLM на занятии 3 — теперь используем для визуала!

NotebookLM: пошагово
Создание презентации
1. Открыть notebooklm.google.com
2. Создать блокнот → загрузить документ (КП, текст, PDF)
3. Слева: панель Studio
4. Нажать «Slide Deck»
5. Настройки: Format, Length, Language
6. Поле «Custom Prompt» — взять промпт с neovida.ai/notebooklm-styles
7. Нажать «Generate»
Пример промпта стиля
Стиль: минимализм, белый фон.
Цвета: акцент #E07461, текст #1A1A2E.
Шрифт: жирный sans-serif для заголовков.
Каждый слайд — одна мысль, макс. 5 пунктов.
Визуализируй цифры как инфографику.

Редактирование: карандаш у слайда → текстовая инструкция
Экспорт: ⋮ → Download PDF / PowerPoint
Gamma: пошагово
Создание со своим стилем
1. Открыть gamma.app
2. Нажать «Create» → Presentation
3. Ввести тему или вставить текст
4. Выбрать тему из галереи (17+ тем)
5. Готово! Редактировать каждый элемент прямо на слайде
Свой стиль (Custom Theme)
Вариант 1: Themes → New Theme
• Colors — фон, акценты (HEX-коды)
• Fonts — заголовки, текст, размеры
• Logo — загрузить свой логотип

Вариант 2: Themes → Import Theme
• Загрузить .pptx файл — Gamma сама извлечёт цвета, шрифты и стиль!

10 правил промптинга для визуала
1
Пиши на английском — кириллицу в кавычках
2
Формула: субъект + стиль + свет + ракурс
3
Описывай что хочешь, а не чего не хочешь
4
Указывай стиль: editorial, product, lifestyle, 3D
5
Управляй светом: golden hour, soft light, dramatic
6
Указывай ракурс: close-up, wide shot, low-angle
7
Будь конкретным — «freckles, auburn hair» вместо «beautiful»
8
Главное — в начало промпта
9
Заземляй абстракции — не «инновация», а конкретная сцена
10
Итерируй — первый результат не финал, уточняй
