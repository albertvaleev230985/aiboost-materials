ВАЖНО: промпт должен уместиться в 500 символов! Включи только самое важное: — Цвета фона и акцентов (HEX-коды) — Тип шрифтов (serif/sans-serif, вес) — Композиция (выравнивание, пустое пространство) — Настроение (1-2 слова)
⚠️
Лимиты: NotebookLM Custom Prompt — до ~2000 символов. Gamma tone/imageStyle — до 500 символов. Поэтому просим ChatGPT уложить промпт стиля в 500 символов — тогда влезет везде.

10 правил промптинга для визуала
1. Пиши на английском — кириллицу в кавычках
2. Формула: субъект + стиль + свет + ракурс + настроение + цвет
3. Описывай что хочешь, а не чего не хочешь (не "no people", а "empty beach")
4. Указывай стиль: editorial photography, product photo, watercolor, 3D render
5. Управляй светом: golden hour, soft natural light, dramatic side lighting
6. Указывай ракурс: close-up, wide shot, low-angle, bird's-eye view
7. Будь конкретным — "freckles, auburn hair" вместо "beautiful"
8. Главное — в начало промпта (модель уделяет больше внимания)
9. Заземляй абстракции — не "инновация", а "команда у голографического дисплея"
10. Итерируй — первый результат не финал: сгенерировал → уточнил → повторил
Формула промпта для изображений
[Объект] + [Сцена] + [Стиль] + [Свет] + [Пропорции] Пример: Создай фотореалистичное изображение для сайта компании по [ваш бизнес]. На фото: [что должно быть на картинке, подробно]. Стиль: профессиональная фотография, мягкий естественный свет. Пропорции: 16:9.

bolt.new — лендинг из промпта
1. Открыть bolt.new → войти через Google
2. Сразу откроется поле для промпта — вставить промпт с websiteprompts.com
3. Или написать: «Создай лендинг для [бизнес]. Секции: герой, преимущества, отзывы, контакты»
4. bolt.new сгенерирует полный проект → превью справа
5. Редактировать в чате: «Добавь калькулятор стоимости»
6. Нажать «Deploy» → получить live-ссылку на Netlify
Free: 1M токенов/мес, безлимитные проекты
v0.dev — лендинг уровня агентства
1. Открыть v0.dev → войти через GitHub/Google
2. В поле «Describe your UI» — вставить промпт или скриншот
3. v0 сгенерирует React + Tailwind код → превью сразу видно
4. Итерировать: «Сделай тёмную тему», «Добавь анимацию при скролле»
5. Или: v0.app/templates → выбрать готовый шаблон → адаптировать
6. Нажать «Deploy to Vercel» → live URL за секунды
Free: $5 кредитов/мес

GoFullPage — скриншот всего лендинга в один клик
1. Установить: Chrome Web Store → поиск «GoFullPage» → нажать «Добавить»
2. Зайти на любой лендинг (например, на Godly или Lapa.ninja)
3. Нажать иконку камеры 📷 в панели расширений (правый верхний угол Chrome)
4. GoFullPage прокрутит всю страницу и сделает один длинный скриншот
5. Скачать как PNG или PDF
6. Этот скриншот → вставить в Lovable как референс для своего лендинга
Без расширения (встроенный способ)
В Chrome: нажать Ctrl+Shift+P (или Cmd+Shift+P на Mac) → написать «Capture full size screenshot» → Enter.
Скриншот всей страницы сохранится в папку «Загрузки».
🔧
