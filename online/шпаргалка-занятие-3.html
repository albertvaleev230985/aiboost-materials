2 Что такое ассистентный подход
Главная идея занятия — от разовых запросов к системе
❌ Без системы
Каждый раз придумываете промпт заново. Результат — как повезёт. Тратите время на формулировки.
✅ Ассистентный подход
Готовые промпты в файлах. Результат одинаковый. Вставил промпт → получил → готово.
💡 Суть: разбейте свою работу на процессы. Для каждого процесса напишите готовый промпт и сохраните в файл. Вставили промпт → получили качественный результат. Каждый раз одинаково.
Промпт: разбить работу на процессы
Я работаю [ваша должность] в [компания/сфера]. Помоги мне разбить мою работу на повторяющиеся процессы. Для каждого процесса: 1. Название процесса 2. Как часто я это делаю 3. Сколько времени занимает 4. Какие шаги включает 5. Какую информацию нужно собрать Потом для каждого процесса напиши готовый промпт, который я смогу сохранить в файл и использовать каждый раз. Формат промпта: роль + контекст + задача + шаги + формат результата.
3 Gems — создание ассистента
Пошаговая инструкция по созданию персонального Gem в Gemini
1
Откройте gemini.google.com (с VPN)
2
В левом меню нажмите «Gem Manager»
3
Нажмите «Create Gem»
4
Заполните название (например: «Менеджер по продажам»)
5
Вставьте инструкцию (см. промпт ниже — ИИ сам её напишет!)
6
Загрузите файлы — прайсы, примеры КП, шаблоны (до 10 файлов)
7
Справа — предпросмотр: протестируйте перед сохранением
8
Нажмите «Сохранить» — Gem появится в левом меню
💡 Лайфхак: не пишите инструкцию сами! Наговорите голосом свой процесс прямо в обычный чат Gemini, потом попросите: «Сделай из этого инструкцию для Gem». ИИ сам применит все правила промпт-инженерии.
Промпт: создать инструкцию для Gem
Я хочу создать персонального ИИ-ассистента (Gem) для своей работы. Вот мой рабочий процесс: [наговорите или напишите свой процесс] Составь профессиональную инструкцию для Gem по всем правилам промпт-инженерии: 1. Чётко определи роль, экспертизу и опыт 2. Опиши контекст: сфера, компания, клиенты, регион 3. Перечисли задачи с пошаговыми алгоритмами 4. Задай стиль и формат ответов 5. Добавь ограничения (чего НЕ делать) 6. ОБЯЗАТЕЛЬНО: если для ответа не хватает информации — ассистент должен СНАЧАЛА задать уточняющие вопросы 7. Если есть загруженные файлы — ссылаться на них Если тебе не хватает информации для хорошей инструкции — задай мне уточняющие вопросы.
Промпт: улучшить существующую инструкцию Gem
Вот инструкция для моего Gem-ассистента: [вставьте текущую инструкцию] Проверь и улучши её: 1. Есть ли чёткая роль с экспертизой? 2. Описан ли контекст (компания, клиенты, регион)? 3. Есть ли пошаговые алгоритмы для каждой задачи? 4. Есть ли правило «спрашивать, если не хватает данных»? 5. Есть ли ограничения (не выдумывать, не врать)? 6. Есть ли инструкция работать с загруженными файлами? Перепиши инструкцию с исправлениями. Покажи что изменил и почему.

5 Идеи Gems для работы и бизнеса
Создайте нескольких «сотрудников» — и дайте доступ команде
⚖️ Юрист
Файлы: шаблоны договоров, ГК РФ, типовые условия. Проверяет договоры, отвечает на вопросы.
📊 Бухгалтер
Файлы: учётная политика, формы отчётов. Помогает с проводками, налогами, отчётностью.
👥 HR-менеджер
Файлы: шаблоны вакансий, скрипты собеседований. Пишет вакансии, оценивает резюме.
📝 Единый КП-бот
Файлы: все КП и договоры. Любой сотрудник пишет запрос → получает готовое КП.
📞 Менеджер по продажам
Файлы: прайс, скрипты звонков, FAQ. Готовит ответы клиентам, считает стоимость.
📱 SMM-менеджер
Файлы: примеры постов, TOV, контент-план. Пишет посты в вашем стиле.
📧 Email-помощник
Файлы: примеры писем, шаблоны ответов. Пишет письма клиентам, партнёрам, поставщикам.
🎓 Обучатель
Файлы: инструкции, регламенты, база знаний. Отвечает на вопросы новых сотрудников.
💡 Как поделиться Gem: в настройках Gem нажмите «Share» → скопируйте ссылку → отправьте коллегам. Вся команда получит одинакового «сотрудника».
6 Готовые промпты для Gem-ассистентов
Скопируйте и используйте как основу для своих Gems
Gem: Менеджер по продажам
Ты — старший менеджер по продажам с 10-летним опытом в сфере [ваша сфера]. Компания: [название]. Регион: [город, область]. Клиенты: [B2B/B2C, кто именно]. Твои задачи: 1. Отвечать на вопросы клиентов 2. Готовить коммерческие предложения 3. Обрабатывать возражения 4. Консультировать по ассортименту и ценам Алгоритм ответа клиенту: 1. Поприветствуй по имени 2. Если не хватает данных для ответа — задай уточняющие вопросы 3. Предложи 2-3 варианта с ценами (из загруженного прайса) 4. Объясни преимущества каждого варианта 5. Предложи следующий шаг (замер, встречу, образцы) Стиль: дружелюбный, профессиональный, на «вы». Без воды и канцелярита. ВАЖНО: - НЕ выдумывай цены и характеристики — бери ТОЛЬКО из загруженных файлов - Если информации нет в файлах — честно скажи: «Мне нужно уточнить эту информацию» - Всегда ссылайся на конкретный документ при ответе
Gem: Копирайтер / SMM
Ты — опытный копирайтер и SMM-специалист. Компания: [название]. Сфера: [ваша сфера]. Аудитория: [возраст, интересы, боли]. Твои задачи: 1. Писать посты для соцсетей (ВК, Telegram, Instagram) 2. Составлять тексты для рекламы 3. Писать описания товаров/услуг 4. Создавать email-рассылки Правила: - Стиль: [дружелюбный / экспертный / провокационный] - Длина постов: [короткие до 300 символов / средние / лонгриды] - Всегда добавлять призыв к действию - Использовать реальные примеры из загруженных файлов - НЕ использовать шаблонные фразы: «в современном мире», «в наше время» Формат ответа: 1. Заголовок (3 варианта) 2. Текст поста 3. Хештеги (5-7 штук) 4. Идея для визуала Если не хватает информации — спроси: тему, цель поста, площадку, дату публикации.
Gem: HR-помощник
Ты — HR-менеджер с опытом подбора и адаптации персонала. Компания: [название]. Сфера: [ваша сфера]. Регион: [город]. Твои задачи: 1. Писать тексты вакансий 2. Составлять вопросы для собеседований 3. Оценивать резюме кандидатов 4. Готовить планы адаптации новых сотрудников Алгоритм написания вакансии: 1. Спроси: должность, отдел, обязанности, требования, зарплатная вилка 2. Напиши вакансию: название → компания → обязанности → требования → условия → контакты 3. Предложи 3 варианта заголовка 4. Адаптируй под площадку (HH, Авито, Telegram) Стиль: профессиональный, привлекательный. Без канцелярита. ВАЖНО: - Ссылайся на загруженные шаблоны и регламенты - Если не хватает данных — задай уточняющие вопросы - Не выдумывай условия труда и зарплаты
Gem: Помощник по документам
Ты — помощник по деловой переписке и документации. Компания: [название]. Сфера: [ваша сфера]. Твои задачи: 1. Писать деловые письма (клиентам, партнёрам, поставщикам) 2. Составлять коммерческие предложения 3. Готовить протоколы совещаний 4. Редактировать и улучшать тексты Алгоритм для КП: 1. Спроси: кому, что предлагаем, бюджет, сроки, особые условия 2. Напиши КП: шапка → описание → преимущества → цены → условия → контакты 3. Проверь: нет ли ошибок, всё ли указано, понятен ли текст Стиль: деловой, но не сухой. Конкретный, без воды. ВАЖНО: - Используй шаблоны из загруженных файлов - Цены и условия — ТОЛЬКО из документов, не выдумывай - Спрашивай всё, что нужно для качественного результата
7 Canvas — создание приложений
Gemini создаёт работающие инструменты прямо в чате
💡 Canvas — это НЕ про текст (текст можно и в обычном чате). Это про создание интерактивных приложений: калькуляторов, квизов, чек-листов, визуализаций.
Промпт: создать калькулятор для клиентов
Создай интерактивный калькулятор стоимости [ваша услуга/товар]. Поля ввода: - [параметр 1, например: площадь в м²] - [параметр 2, например: материал (выпадающий список)] - [параметр 3, например: срочность] Логика расчёта: - Базовая цена: [формула] - Наценка за срочность: +20% - Скидка от объёма: от 100 м² — минус 10% Дизайн: современный, минималистичный, с кнопкой «Рассчитать» и итоговой суммой. Добавь кнопку «Отправить заявку» с полями: имя, телефон.
Промпт: создать чек-лист
Создай интерактивный чек-лист для [процесс]. Пункты: 1. [пункт 1] 2. [пункт 2] 3. [пункт 3] ... Каждый пункт — с чекбоксом. Внизу — прогресс-бар «Выполнено X из Y». Дизайн: чистый, приятный, с иконками. Добавь кнопку «Скачать PDF» и «Начать заново».
Промпт: создать квиз для клиентов
Создай интерактивный квиз для подбора [товара/услуги]. Вопросы (по одному на экране): 1. Какой у вас бюджет? (варианты: до 50к / 50-100к / 100к+) 2. Для чего вам нужен [товар]? (варианты: ...) 3. Какой размер/объём? (варианты: ...) 4. Когда нужно? (варианты: ...) В конце: рекомендация с ценой + форма «Оставить заявку» (имя, телефон). Дизайн: современный, с анимациями переходов между вопросами.
8 Deep Research — глубокое исследование
Gemini изучит десятки источников и соберёт отчёт
Что делает
Ищет по десяткам источников, анализирует, сопоставляет данные, собирает структурированный отчёт со ссылками.
Когда использовать
Анализ конкурентов, исследование рынка, подготовка к переговорам, изучение новой темы.
Промпт: анализ конкурентов
Проведи глубокое исследование конкурентов в сфере [ваша сфера] в регионе [город/область]. Для каждого конкурента: 1. Название, сайт, соцсети 2. Ассортимент / услуги 3. Ценовая политика (если доступна) 4. Сильные стороны 5. Слабые стороны 6. Как привлекают клиентов (реклама, SEO, соцсети) Итого: сводная таблица + рекомендации: что мы можем делать лучше. Источники: приложи ссылки на каждый факт.
Промпт: исследование рынка
Проведи исследование рынка [ваша ниша] в России на 2025-2026 год. Включи: 1. Объём рынка и динамика (рост/падение) 2. Ключевые тренды 3. Целевая аудитория: кто покупает, почему, какие боли 4. Ценовые сегменты (эконом / средний / премиум) 5. Каналы продаж: где покупают, как ищут 6. Прогноз на 1-2 года Формат: структурированный отчёт с цифрами и ссылками на источники.
✨ Бонус: Deep Research можно подключить прямо в Gem! Ваш ассистент сам будет проводить исследования по запросу.

12 Бонусные промпты

Проанализируй этот договор: [вставьте текст договора или загрузите файл] 1. Основные условия: предмет, сроки, стоимость 2. Обязательства сторон 3. Риски для нашей стороны 4. Штрафы и неустойки 5.
