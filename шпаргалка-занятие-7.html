Агент — что это
Главная мысль урока
🎯
Агент = несколько LLM-моделей (ChatGPT, Claude, Gemini) + оболочка из инструментов, памяти, планировщика и скиллов. Он сам планирует шаги и выполняет задачи.

Бот = кнопки, «если А → делай Б». Ассистент = интеллект для одной задачи (GPTs, Gems). Агент = цепочка шагов, где ИИ сам решает что делать.

📝 Пошагово: создаём агента в Suvvy
1
Открываем suvvy.ai → «Попробовать бесплатно» → регистрация через email
2
В дашборде → «Создать агента» → шаблон «Продажник»
3
Имя и роль агента. Пример: «Анна, менеджер по навесам»
4
Загружаем базу знаний: прайс, FAQ (частые вопросы), текст о бизнесе
5
Раздел «Сценарии» → цель: «Квалифицируй клиента, собери 3 параметра, предложи замер»
6
Интеграция с Telegram: создаём бота через @BotFather (команда /newbot), вставляем токен (секретный ключ) в Suvvy
7
Тест в песочнице (тестовый чат Suvvy) → пишем вопрос клиента → смотрим ответ
8
Если ответ плохой → правим базу знаний или инструкцию. Это тюнинг (донастройка)

Мультиагентность — 3 способа
Когда один агент не справляется — несколько работают вместе
1. Директор + сотрудники (иерархия)
Один агент раздаёт задачи другим, собирает результат. Пример: маркетинговая кампания — 1 планирует, 3 делают.
2. Эстафета (swarm)
Нет начальника. Каждый передаёт задачу следующему: парсер → аналитик → отправка. Пример: собрать данные → анализ → отчёт по почте.
3. Дебат (генератор vs критик)
Два агента спорят: один предлагает, другой ищет ошибки. Итог — проверенный результат. Пример: юрист пишет договор, ревизор проверяет.
